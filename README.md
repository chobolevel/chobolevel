## 🚀 Featured Projects

### [Logging SDK](https://github.com/chobolevel/logging)
Spring Boot 애플리케이션에 **의존성 하나로 끼워 넣을 수 있는** 커스텀 로깅 라이브러리

`Logback AppenderBase` 확장 기반으로 직접 설계한 SDK입니다. 단순 사용을 넘어 라이브러리 레벨에서 확장 가능한 구조(인코더 / 어펜더 플러그인 방식)를 직접 고민하고 구현했습니다.

`Java` `Logback` `Spring Boot AutoConfiguration`

---

### [초로 (chobolevel)](https://github.com/chobolevel/log)
개발 학습 내용을 기록하고 실제 서비스 운영 환경을 직접 구축해본 **개인 기술 블로그 플랫폼**

인증 시스템, 실시간 채팅, 파일 업로드부터 CI/CD와 모니터링까지 — 기능 구현을 넘어 서비스 운영에 필요한 인프라까지 직접 설계하고 구축했습니다.

- JWT 기반 인증/인가, WebSocket(STOMP) 실시간 채팅, S3 Presigned URL 파일 업로드
- Prometheus + Grafana + Loki 기반 모니터링 환경 구축, Discord Webhook 오류 알림
- API Warm-up 로직으로 초기 응답 시간 **3초 → 500ms** 개선
- GitHub Actions + Docker + Jib 기반 CI/CD 파이프라인 구축

`Kotlin` `Spring Boot` `WebSocket` `Flyway` `Docker` `Prometheus` `Grafana`
